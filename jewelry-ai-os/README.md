# AuraOS — Jewellery AI Automation OS

A premium, single-page marketing website for **AuraOS**, an AI automation
operating system built for jewellery brands. It showcases the possibilities
and areas of AI automation & integration that help jewellery businesses
**increase sales and revenue**.

Built by **SignQube Innovations**.

## What it covers

The site presents 12 automation modules, each mapped to a revenue outcome:

| Module | Area of automation | Revenue impact |
| --- | --- | --- |
| AI Sales Concierge | 24/7 WhatsApp / Instagram / web chat | More enquiries converted |
| Virtual Try-On & AR | Product visualisation | Fewer returns, confident buyers |
| Auto Catalogue Studio | AI photography & SEO copy | 9× faster listing |
| Personalised Recommendations | Upsell / cross-sell | Higher order value |
| Live Gold-Rate Pricing | Real-time quotations | Accurate instant quotes |
| Cart & Lead Recovery | Drop-off win-back | Recover lost orders |
| Campaign & Ads Automation | Festival / season marketing | Better ROAS |
| Visual & Voice Search | Frictionless discovery | More conversions |
| Inventory & Demand Forecasting | Stock optimisation | Less dead stock |
| CRM 360 & Loyalty | Retention automation | More repeat buyers |
| AI Design Studio | Custom / bespoke design | High-margin orders |
| Reviews & Reputation | Trust automation | More social proof |

It also includes: how-it-works flow, revenue-impact stats, audience
segments, pricing tiers, FAQ and contact CTA.

## Tech

- A single, self-contained `index.html` — no build step, no dependencies.
- Vanilla HTML + CSS + a little JavaScript (scroll reveals, count-up stats,
  seamless marquee, sticky nav, mobile menu).
- Fully responsive, dark "luxury gold" theme.

## Run locally

Just open the file:

```bash
open index.html        # macOS
# or
python3 -m http.server # then visit http://localhost:8000
```

## Deploy

Any static host works (Vercel, Netlify, GitHub Pages, Cloudflare Pages).
Point the host at the repository root and serve `index.html`.
